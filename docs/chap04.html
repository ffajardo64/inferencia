4  Estimação pontual

Assumiremos uma amostra aleatória \(X_1,X_2,\ldots,X_n\) de uma população com densidade \(f(\cdot;\theta)\), onde a forma da densidade é conhecida, mas o parâmetro \(\theta\) é desconhecido. Em alguns cenários \(\theta\) será um vetor de parâmetros \(\theta=(\theta_1,\theta_2,\ldots,\theta_m)\). Denotaremos \(\Theta\) o espaço paramétrico, i.e. o espaçoo de todos os possíveis valores de \(\theta\)

Definição 4.1 (Estimador) Seja \(X_1,X_2,\ldots,X_n\) uma amostra aleatória de uma população com função de densidade \(f(\cdot;\theta)\). Um estimador \(T=t(X_1,X_2,\ldots,X_n)\) é uma estatística usada para estimar alguma função de \(\theta\).

Exemplo 4.1 A estatística \(\bar{X}_n=\frac1n\sum_{i=1}^nX_i\) é um estimador da média \(\mu=\mathbb{E}[X]\).

4.1 Propriedades dos estimadores

Definição 4.2 (Viés) Um estimador \(T=t(X_1,X_2,\ldots,X_n)\) se diz não-viesado do \(\tau(\theta)\) se e somente se \(\mathbb{E}[T]=\tau(\theta)\), para todo \(\theta\in\Theta\). O viés do estimador \(T\) é dado por \(B(T)=\mathbb{E}[T]-\tau(\theta)\).

Seja \(X_1,X_2,\ldots,X_n\) uma amostra aleatória de uma população com função de densidade \(f(\cdot;\mu,\sigma^2)\). Sejam \(\bar{X}_n=\frac1n\sum_{i=1}^nX_i\) e \(S^2=\frac{1}{n-1}\sum_{i=1}^n(X_i-\bar{X}_n)^2\) os estimadores de \(\mu\) e \(\sigma^2\), então

\(\mathbb{E}[\bar{X}_n]=\mathbb{E}\left[\frac1n\sum_{i=1}^nX_i\right]=\frac1n\sum_{i=1}^n\mathbb{E}[X_i]=\mu\);

\[\begin{align*} \mathbb{E}\left[S^2\right]&=\mathbb{E}\left[\frac{1}{n-1}\sum_{i=1}^n(X_i-\bar{X}_n)^2\right]\\ &=\frac{1}{n-1}\sum_{i=1}^n\mathbb{E}\left(X_i-\bar{X}_n\right)^2 =\frac{1}{n-1}\sum_{i=1}^n\mathbb{E}\left(X_i-\mu+\mu-\bar{X}_n\right)^2\\ &=\frac{1}{n-1}\sum_{i=1}^n\left\{\mathbb{E}\left(X_i-\mu\right)^2+\mathbb{E}\left(\mu-\bar{X}_n\right)^2-2\mathbb{E}\left(X_i-\mu\right)\left(\mu-\bar{X}_n\right)\right\}\\ &=\frac{1}{n-1}\left\{n\sigma^2-\sigma^2\right\}=\sigma^2. \end{align*}\] Daí, \(\bar{X}_n\) e \(S^2\) são esimadores não-viesados de \(\mu\) e \(\sigma^2\), respectivamente.
